Currency rates from exchangerates

This module updates currency rates for all enabled currencies. It uses ExchangeRates API to update currency rates.

Installation

  • Go to Apps.

  • Search Module For: aspl_currency_update.

  • Click Install.

  • Get an API key from from exchangerates.

  • Go to Settings -> Accounting.

  • Set API key under Currencies -> Currency Rate Update (Exchange Rates)

Dependencies

Screenshots

Specify API key in accounting configuration.

Enable currencies you want to work with.

It syncs latest values of the currency by date.
